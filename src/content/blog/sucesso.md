# **What is DecapBridge**

DecapBridge is an authentication and user management service designed specifically for Decap CMS, aiming to make it as easy as possible for non-technical users to login and start editing content on your website.



As a site owner, you can simply invite collaborators by email, then getting started is a familiar self-service from their end. They can choose to login with Google, Microsoft or set a password. If they forget their password, they can reset it as well, without any intervention from your part. You can then view who has access to edit your sites and then remove their access if needed.



This project is not to be confused with Decap CMS which is a distinct open source project that does not include a user management and authentication service.
